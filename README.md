py2km
=====
Pinyin 2 Kwik Mandarin converter

# Feature
* Pinyin (Tone Number) to Pinyin (Tone Marks) or Kwik Mandarin
* Pinyin (Tone Marks) to Kwik Mandarin
* Hanzi to Pinyin (Tone Marks) or Kwik Mandarin
* Hanzi to Dictionary Mode
* Pinyin tone rules
* Printable
